# krissanders.com updates feed

This repo hosts public content files that are pulled into https://krissanders.com/ at runtime.

## Files

### availability.json
Controls the Engagement box (previously “Availability”).

### status.txt
Controls the Status box. Add/remove lines freely — the box expands naturally.

### selected-work.json
Controls the Selected Work section. Each item renders as a dropdown.

### operating-principles.txt
Controls the Operating Principles card.

### meta.json
Provides a Last Updated trust signal displayed in the site footer.

## Raw base URL used by the site
`https://raw.githubusercontent.com/Ksnarf/site.1.updates/main`
